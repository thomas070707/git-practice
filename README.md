# gradebook

과목별 점수를 입력받아 성적을 관리하는 프로그램입니다.

## 기능
- 과목별 점수 입력
- 평균 점수 계산
- 성적 결과 출력
- 최고점 / 최저점 과목 확인

## 실행 예시
과목명: 국어 → 88점
과목명: 영어 → 92점
...
==============================
       성적 결과
==============================
  국어           88.0점
  영어           92.0점
  수학           75.0점
  과탐           81.0점
------------------------------
  평균           84.0점
==============================

최고점: 영어 (92.0점)
최저점: 수학 (75.0점)

## 사용법
python grades.py
